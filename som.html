fastsom.som¶

SOM Model¶

Som¶

class fastsom.som.som.Som(size: Tuple[int, int, int], alpha=0.3, dist_fn: Callable = <function pdist>, neigh_fn: Callable = <function neigh_gauss>, neigh_diff_fn: Callable = <function neigh_diff_standard>)[source]¶

Self-Organizing Map module.

Parameters

  • size (SomSize2D) – The 2D map size: (rows, cols, n_features)

  • alpha (float default=0.3) – The initial alpha hyperparameter. This is usually overwritten by a SomTrainer.

  • dist_fn (Callable = pdist) – The distance function used to compute records-to-codebook distances.

  • neigh_fn (Callable = neigh_gauss) – The neighbourhood scaling function.

  • neigh_diff_fn (Callable = neigh_diff_standard) – The neighbourhood difference function. Can be standard or toroidal.
