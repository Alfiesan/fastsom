SOM Training interpretation¶

class fastsom.interp.interp.SomInterpretation(learn)[source]¶

SOM interpretation utility.

Displays various information about a trained Self-Organizing Map, such as topological weight distribution, features distribution and training set distribution over the map.

Parameters

learn (SomLearner) – The learner to be used for interpretation.

Shows a heatmap for each feature displaying its value distribution over the codebook.

Parameters

  • recategorize (bool default=True) – If True, converts back categorical features that were previously made continuous.
